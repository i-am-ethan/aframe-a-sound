# How to make a audio in A-FRAME

A-FRAME に audio を組み込む最も単純な方法です。

# Usage

a-sound 要素

```bash
<a-sound
        src="src: url(bag.mp3)"
        autoplay="true"
        position="0 2 5"
      ></a-sound>
```
